WebService::Gyazo
=================
WebService::Gyazo - perl image upload library for gyazo.com

 
INSTALLATION
 
To install this module type the following:
 
	perl Makefile.PL
	make
	make test
	make install
 
DEPENDENCIES
 
This module requires these other modules and libraries:
 
	LWP
	LWP::UserAgent
	LWP::Protocol::socks
	HTTP::Request::Common
	URI::Simple
